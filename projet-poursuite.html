Application de poursuites d'études

Site web de gestion académique et administrative

PHP / Symfony JavaScript SQL HTML/CSS
Période : 2024 Équipe : Projet académique Contexte : BUT Informatique

Contexte du projet

Ce projet a été développé dans le cadre de mon BUT Informatique. Il s'agit d'un site web complet destiné à la gestion académique et administrative d'une formation universitaire. La plateforme permet de gérer efficacement les étudiants, leurs résultats, leurs vœux de poursuite d'études, ainsi que les différents rôles et permissions des utilisateurs.

Fonctionnalités principales

  • Gestion complète des étudiants : Opérations CRUD (Create, Read, Update, Delete)
  • Suivi académique : Suivi des résultats et performances des étudiants
  • Système de vœux : Gestion des poursuites d'études et candidatures
  • Contrôle d'accès : Gestion des rôles et permissions utilisateurs
  • Import/Export : Traitement de fichiers CSV et Excel pour les données
  • Rapports et statistiques : Génération de tableaux de bord analytiques
  • Interface d'administration : Panneau de contrôle complet pour les administrateurs
