Wavy Météo

Application web dédiée aux pratiquants de sports nautiques

PHP / Symfony SQL API Météo JavaScript HTML/CSS
Période : Depuis septembre 2025 Équipe : 6 personnes Méthode : Agile

Présentation du projet

Wavy Météo est une application web multi-plateforme conçue pour les passionnés de sports nautiques. Elle permet de consulter les données météorologiques essentielles (vent, vagues, marées, température) pour pratiquer des activités comme le surf, la voile, le kitesurf, la plongée ou simplement profiter de la plage, et ce partout dans le monde.

Ce projet a été réalisé en équipe de 6 personnes dans le cadre de notre formation BUT Informatique, en suivant la méthodologie Agile pour garantir une organisation efficace et une livraison itérative des fonctionnalités.

Fonctionnalités principales

  • Données météo en temps réel : Affichage des conditions météorologiques essentielles aux sports nautiques
  • Couverture mondiale : Recherche de spots et de conditions météo partout dans le monde
  • Interface personnalisable : Adaptation aux préférences utilisateurs (thèmes, unités, favoris)
  • Accessibilité : Options pour les personnes en situation de handicap
  • Authentification sécurisée : Système de connexion avancé avec protection des données
  • Gestion des accès : Contrôle des permissions et rôles utilisateurs

Défis techniques relevés

  • Intégration multi-API : Gestion de plusieurs sources de données météo

  • Accessibilité WCAG : Respect des normes pour une application inclusive
  • Sécurité renforcée : Protection contre les injections SQL, XSS et CSRF
